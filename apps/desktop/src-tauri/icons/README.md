# Icon Files

This directory should contain the following icon files for the application:

- `32x32.png` - 32x32 pixels icon
- `128x128.png` - 128x128 pixels icon
- `128x128@2x.png` - 256x256 pixels icon (for Retina displays)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

For now, these are placeholder files. Replace them with actual EvorBrain icons before release.
